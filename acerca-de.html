Acerca de nosotros

¿Quienes somos?

Somos un equipo de la Universidad Autónoma de Nuevo León Facultad de Ingeniería Mecánica y Electrica cursando el primer semestre de la universidad, estamos desarrollando una página web para realizar lo aprendido durante el semestre.

Esta página web fue realizada en (HTML) que significa Hyper Text Markup Language que se utiliza para dar estructura y propiedades a páginas web las cuales después de ser modificadas en CSS podrán ser subidas a la web. Por otra parte esta (CSS) que significa hojas de estilo en cascada y es utilizado para darle estilo a hojas de html y modificar sus elementos para hacerlos mas amenos a la vista
